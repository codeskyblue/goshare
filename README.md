goshare
=======

share file in current machine.

## Installation
```
go get -v github.com/codeskyblue/goshare
```

## Usage
Start a file server and Generate download link address.

Just one command.

```
$ goshare some_file.txt
http://10.240.171.86:10351/0aa04a5d/some_file.txt
```
